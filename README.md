## 📖 About

This is a **minimal todo list application** I built while learning Vue.js basics. Nothing fancy, just the classic todo example to understand Vue's core concepts!

> 🧪 **Learning Purpose Only**  
> My first Vue experiment - simple, clean, and perfect for beginners!

---

## ✨ Features

| Feature | How It Works |
|---------|--------------|
| **✅ Add Tasks** | Type and hit enter or click button |
| **❌ Delete Tasks** | Remove tasks you don't need |
| **✓ Mark Complete** | Click to toggle done/undone |
| **📊 Task Counter** | Shows how many tasks left |

---

## 🚀 Quick Start

```bash
# 1. Clone it
git clone https://github.com/yourusername/vue-todo.git
cd vue-todo

# 2. Open in browser
open index.html  # or just double-click it!
```

**That's it!** No build tools, no complex setup - just Vue CDN and vanilla JS.
